写在前面：个人常用的一些实用高效的办公软件汇总

浏览器

  • Chrome

    谷歌浏览器很好用，有很多强大的插件，但是很大的缺点是太耗内存

  • Microsoft Edge （windows自带）

    新版Edge搭载Chrome内核，是Chrome的良好替代

  • Firefox

    Linux下的专业浏览器

办公

  • Office：

    Windows自带，Office全家桶办公必备，个人觉得比WPS好用

  • BandZip

    解压缩神器，支持各种解压缩格式，轻量

  • docsmall

    通通变小之压缩神器

  • Typora

    非常好用Markdown编辑器，侧重于编辑，但是管理有待提升

  • Matlab

    强大的数学科研工具，不做评价，一个字非常牛，但是太过臃肿，可以用python替代

  • Octave

    开源的Matlab，且更加轻量

  • 格式工厂

    格式转换神器

  • OBS Studio

    视频录制软件

  • 截图

    一般社交软件都自带截图快捷键，如QQ，微信等，独立截图软件反而没有必要，快捷键Ctrl+Alt+A

    当然也有好用的截图工具：ShareX，截图、处理截图、传送截图，shareX 给你包圆了。

  • OriginLab

    学术统计制图软件

  • Visio

    专业绘图软件

  • Endnote

    文献整理软件，写论文必备，一般也可用NoteExpress

  • 知云翻译

    英文文献的阅读的福音，英语菜鸡科研者者必备

  • pdf24 tools

  • LANdrop

    局域网跨平台传输文件利器

设计

  • Adobe

    一般下面列出的前三个软件就够用了，个人使用需要破解，商用建议买正版

    • PhotoShop
    • Premiere
    • After Effect
    • Audition
    • …………

  • Autodesk

    ​ 3D建模软件，个人使用需要破解，商用建议买正版

    • 3DMax

    • Maya

    • Inventor

  • Solidworks

    强大的工业级3D建模软件

  • HitFilm

    强大的电影剪辑工具

  • PotPlayer

    万能视频播放器

​

  • billfish

    高效的素材管理工具

  • Arctime Pro

    一款简单、强大、高效的字幕制作软件，完全免费，还支持压缩视频

效率

  • Listary

    强大的全局搜索器

  • Everything

    和Listary类似，个人觉得Listary比Everything更好用

  • Wox

    开源的Listary

  • uTools

    一切皆插件，各取所需

  • Quicker

    鼠标启动器，效率神器

  • 版本控制：Git，Gitkraken

    • Git

      写代码的应该没有不知道的吧

    • GitKraken

      git可视化神器，告别命令行

  • Wgesture

    全局鼠标手势

  • Ditto

    剪切板管理工具

  • Fences

    桌面整理管理工具

  • MaxTo

    桌面分屏工具

开发者工具

  • Visual Studio Code

    开源，最强IDE，插件无数

  • Android Studio

    安卓开发专用IDE

  • Python开发：PyCharm，Jupyter Notebook

    • PyCharm

      社区版免费

    • Jupyter Notebook

      下载Anaconda携带

  • Java开发：IDEA

    社区版免费，Java开发最舒服的IDE

  • Web开发：WebStorm

    个人使用需要自行去网上找激活码

  • C/C++开发：Microsoft Visual Studio，Dev C++

    • Visual Studio 2019

    • Dev C++

      适合初学者，简单容易上手

  • Arduino

    Arduino开发

专业开发工具

  • ArcGIS

    地理测绘工具

  • 电路仿真软件

    • LabView

    • Multisim

  • Wireshark

    网络抓包工具

  • Vivado

    FPGA设计软件

  • Xshell

    终端工具

  • Xftp

    网络文件传输工具

  • Winhex

    十六进制编辑器

运维测试

  • Docker

    容器环境搭建

  • Vmware

    虚拟机

清洁整理

  • geek

    最强卸载工具，超级轻量

  • CCleaner

    垃圾清理工具，轻量

  • 火绒

    杀毒软件，其实windows自带的就够了

  • Dism++

    轻量级电脑管家

休闲娱乐

  • AIMP

    强大的音乐播放器

  • PotPlayer
