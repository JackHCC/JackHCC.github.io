插件推荐

  • AdBlock
    • 最佳广告拦截工具

  • FeHelper(前端助手)
    • JSON自动格式化、手动格式化，支持排序、解码、下载等

  • ColorPick Eyedropper
    • 好用的取色工具，A zoomed eyedropper & color chooser tool that allows you to select color values from webpages and more.

  • Convertio
    • 在线将文件从一种格式转换为另一种格式. 支持超过 2500 种不同的转换。

  • Global Speed
    • 视频速度控制，网页视频播放速度支持任意速率

  • Google学术搜索按钮
    • 可让您在浏览网页时查询学术文章。

  • LeetCode Helper by labuladong
    • 力扣辅助刷题插件，便于 labuladong 的读者高效刷题。LeetCode刷题神器！

  • Octotree - GitHub code tree
    • Github文件树形结构展示，GitHub on steroids

  • Proxy SwitchyOmega
    • 轻松快捷地管理和切换多个代理设置。

  • SuperCopy 超级复制
    • 一键破解禁止右键、破解禁止选择、破解禁止复制、破解禁止粘贴，启用复制，启用右键，启用选择，启用粘贴。

  • Tampermonkey油猴
    • 最强大的插件管理器，The world’s most popular userscript manager，yyds，不多说，自己体会！

  • Web Scraper - Free Web Scraping
    • 网络爬虫助手，Web data extraction tool with an easy point-and-click interface for modern web

  • 哔哩哔哩下载助手
    • 哔哩哔哩视频下载助手, B站下载助手

  • 沙拉查词-聚合词典划词翻译
    • Saladict 沙拉查词是一款专业划词翻译扩展，为交叉阅读而生。

  • 类似的网站 - 发现相关网站
    • 立即发现与您当前正在浏览的网站类似的网站。探索网络，了解更多信息。

  • 青柠起始页
    • 一个简洁、美观、实用的浏览器起始页。此为青柠起始页官方扩展。大道至简！
